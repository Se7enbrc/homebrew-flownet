# Homebrew Tap for FlowNet

Official Homebrew tap for [FlowNet](https://github.com/se7enbrc/flownet) - AWDL suppression daemon for optimized WiFi performance on macOS.

## Installation

```bash
brew tap se7enbrc/flownet
brew install flownet
```

## Usage

See the [FlowNet README](https://github.com/se7enbrc/flownet#readme) for full documentation.
